# code_dhtlight_mobius
Mobius 대회 출전을 위한 교육용 예제 코드입니다.

DHT 센서 및 조도 센서 데이터를 수집하여  
Mobius(oneM2M 플랫폼)로 전송하는 구조를 학습하기 위해 제작되었습니다.

## 📌 주요 기능
- DHT 센서 온·습도 데이터 수집
- 조도 센서 데이터 수집
- Mobius로 CIN 생성 및 전송
- MQTT/HTTP 기반 통신 구조 이해

## 🛠 사용 기술
- Raspberry Pi
- Python
- Mobius (oneM2M)
- MQTT / HTTP
